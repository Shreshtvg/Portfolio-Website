# Shresht V G — Software Engineering Portfolio

An interactive, responsive, and beautifully styled full-stack portfolio designed for **Shresht V G**, built using **React 18**, **Vite**, **TypeScript**, **Tailwind CSS**, and **Framer Motion** (via `motion/react`).

This website features a high-contrast dark digital theme, smooth page transitions, interactive visual elements, modular architectures, and dynamic contact integration.

https://shreshtvgportfolio.vercel.app/

---

## 🚀 Key Features

- **Cosmic Dark Digital Theme**: Highly polished, modern dark theme designed with generous negative space, sleek borders, and custom color accents.
- **Dynamic Greeting**: Rotating developer roles, a persistent **Connectivity** panel with instant socials access, and copyable email utilities.
- **Interactive Projects Grid**: Displays featured projects (including *Hospital FrontDesk System*, *Convo Chat Platform*, *School Management System*, and *RAG-based PDF QA*) with detail modals and live hyperlinks.
- **Interactive Experience & Journey Timeline**: Showcases internships (Manhattan Associates, CCBD Center PESU, WeCreateProblems) with intuitive expand/collapse cards.
- **Refined Contact Flow & Integration**: A validation-backed contact form that packages message parameters securely and redirects users to preloaded mail drafts with a single click, overcoming typical in-app email dispatch issues.

---

## 🛠️ Technology Stack

- **Framework**: [React 18](https://react.dev/) + [Vite](https://vite.dev/) (Fast development cycle, zero-configuration build engine)
- **Language**: [TypeScript](https://www.typescript.org/) (Strict type-safety, interface-based data layer)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Utility-first, responsive, lightweight layout classes)
- **Animations**: [Framer Motion / Motion](https://motion.dev/) (Fluid micro-interactions, spring layouts, stagger lists)
- **Icons**: [Lucide React](https://lucide.dev/) (Sleek, minimalist vector designs)
